# atn-cli
This is a React scaffolding

## Available Scripts

In the project directory, you can run:

### `npm install` or `yarn`
Clone project on local and install dependencies for project needs.

### `npm link`

### `npm run watch`
Run the project locally.

## Project Configurable Command

### `atn init react-tempalte [file name]`
A project named [file name] is initialized locally by the template.

### `atn config get`
Gets the configuration of the local .atnrc file.

### `atn config set registry [template name]`
Sets the name of the [template name] to be pulled locally.

### `atn -V` or `atn --version`

### `atn -h` or `atn --help`
